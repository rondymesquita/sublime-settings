# My Sublime Text 3 Settings

## Windows
```bash
 cd "$APPDATA/Sublime Text 3/Packages/User"
```

## Linux
```bash
 cd ~/.config/sublime-text-3/Packages/User
```

## MacOS
```bash
 cd ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/User/
```

# Setup

git init
